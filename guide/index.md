## 注册账号

要开始游玩未界音律UNTiTlED，您首先需要注册一个账号。

::: warning
**"以离线模式游玩"** 仅限在**无网络连接**的情况下游玩。由于本地终端不自动包含任务文件，所以您首先需要一个 UNTiTlED 账号来下载任务文件。
:::

:::tabs
== 网页注册
您可以在[这里](https://untitledrhythm.online/login)注册并登录UNTiTlED账号。

1. 点击下方的 **“立即注册”** 按钮。
2. 输入您的用户名、邮箱地址和密码。
3. 点击 **“注册”** 按钮。

== 游戏内注册
或者，您也可以在游戏中注册账号。

1. 点击右上方的账户中心，选择 **“创建新的UNTiTlED账户”**
2. 输入您的用户名、邮箱地址和密码。
3. 点击 **“提交”** 按钮。
:::

**注册成功后，您还需要验证您的邮箱地址。**
请在7日内检查您的邮箱，点击其中的验证链接，完成验证。
如果没有找到验证邮件，请检查您的垃圾邮件文件夹。

## 登录账号

:::tabs
== 网页登录
您可以在[这里](https://untitledrhythm.online/login)登录UNTiTlED账号。

1. 输入您的用户名或邮箱地址。
2. 输入您的密码。
3. 点击 **“登录”** 按钮。

== 游戏内登录
或者，您也可以在游戏中登录账号。

1. 点击右上方的账户中心，选择 **“登录UNTiTlED账户”**
2. 输入您的用户名和密码。
3. 点击 **“登录”** 按钮。
:::

登录成功后，您便可以可以访问 未界UNTiTlED 了。

在游戏中，单击 **“初始化访问”** 按钮，即可进入主菜单界面。
## 剧情模式

## 在线模式

在主菜单界面，单击 **“在线任务”** 按钮，即可进入在线任务界面。

其允许您访问 未界UNTiTlED 的全部在线任务，您可以选择您感兴趣的任务，并下载对应的任务文件。

单击右下方的 **”本地“** 按钮，即可进入本地任务界面。

## 本地模式

在主菜单界面，单击 **“本地任务”** 按钮，即可进入本地任务界面。

其允许您访问下载的全部本地任务，您可以选择您感兴趣的任务，开始执行。

::: tip
在开始执行您的首个任务前，强烈推荐您首先完成 **”任务教程“** ，以了解任务的基本操作。

单击左侧的 **”任务教程“** 按钮，即可进入任务教程。
:::

## 任务结算

在任务成功或失败后，您将会进入任务结算界面。

只要您处于在线模式，并且任务难度状态正确，无论任务是否成功，您的任务结果都会被上传到服务器。

若上传失败，您可以单击右下角的 **”重新上传“** 按钮，重新上传任务结果。

单击左下方的 **“重试”** 按钮，即可重新执行任务。

单击右下方的 **“继续”** 按钮，即可返回任务列表界面。
