# 未界音律UNTiTlED平台版权政策

## 简介

本平台包含以社区创作（UGC）为核心的音乐游戏服务。部分谱面（beatmap）与封面由社区用户上传，上传者应当对其提交的内容拥有相应的版权或已取得必要的授权。平台不会以直接出售用户上传作品为主要盈利方式。

我们重视版权与其他知识产权。如需与平台洽谈音乐/曲目商业授权，请联系：**【[UMARU_feedback@163.com](mailto:UMARU_feedback@163.com)】**。

本平台采用下列版权处理政策（参照并遵循 DMCA / 通用 notice-and-takedown 流程）：

* 在有合理理由认为某内容可能侵犯第三方版权的情况下，会**迅速阻止访问或移除**该内容；
* 对多次经核实的侵权者将**移除其内容并终止服务**（封禁账号等）。

## 如何举报版权侵权（提交下架通知）

若您认为本平台上某项内容侵犯了您的版权，请将 **完整的侵权声明**（下述信息）发送至本平台的指定接收邮箱（Designated Agent）。为便于我们快速处理，请务必包含下列内容：

1. **声明签名** —— 授权人签名（实体或电子签名均可），即有权代表版权所有人的自然人或法人签署；
2. **被主张侵权的作品识别** —— 明确指出您主张被侵权的作品（作品名、作者、作品登记号或其他能识别该版权的资料）；
3. **侵权内容定位** —— 明确指出涉嫌侵权的具体平台资源（例如：谱面名称 / 封面使用地点 / 具体页面 URL 等），并提供足够信息以让平台定位该资源；
4. **权利人联系方式** —— 提供权利人或代理人的姓名/公司名、邮寄地址、电话号码及电子邮件地址；
5. **善意声明** —— 声明您有理由相信被投诉的使用行为未经版权人授权、其代理人或法律允许；
6. **属实与授权声明** —— 在经法律允许的范围内，声明您所提供信息真实准确，并在宣誓或伪证责任下确认您有权代表版权人提出该通知。

**（示例短文本）**：

> 我，[权利人姓名]，在此郑重声明：我有权对作品《*****》主张版权。本人特此声明所述信息真实准确，且在伪证责任下确认本人已授权提交本下架通知。签名：*****；日期：_____。

> 提交到：**【[UMARU_feedback@163.com](mailto:UMARU_feedback@163.com)】**

## 我们收到完整通知后将采取的措施

在收到**完整且形式合规**的版权下架通知后，本平台将会依照既定流程处理，通常包括但不限于：

1. **移除或断开对该涉嫌侵权内容的访问**；
2. **通知被投诉的上传者 / 内容提供者**（并在合理期限内告知其如何提交反通知/申诉）；
3. **记录并保存处理记录与证据**（包括原始通知、处理时间、相关文件哈希、IP、上传日志等）；
4. **在必要时对多次侵权者采取进一步措施**（例如永久删除其内容、封禁账号等）；
5. 若涉及特定法域（如中国大陆），我们还将按照适用法律要求**转送通知或在平台公告**。

> 注：平台并非版权归属仲裁机构；若双方在移除后通过反通知进入争议程序，平台会依据法律与流程继续执行（见下文反通知说明）。

## 反通知（Counter-Notice / 申诉）

若您（被移除内容的上传者 / 内容提供者）认为被下架/禁用的内容并非侵权，或您已获得版权人授权，您可以向我们提交**反通知**，反通知应包含下列信息：

1. 被移除内容的**明确识别信息**（被移除内容原先所在的 URL / 资源 ID 等）；
2. 您的**签名**（实体或电子签名）；
3. 一份书面声明，内容包括：您真诚地相信该内容是因错误或误认而被移除；并在伪证责任下声明该信息真实；
4. 您的联系信息（姓名/公司、地址、电话、电子邮件）；
5. 您同意接受**适用司法管辖**（请在此条中填写平台指定的法院或仲裁地，或按双方约定填写）。

**（示例短文本）**：

> 我，[上传者姓名]，声明我有权发布被下架的内容（或我有合理理由相信该内容并不构成侵权）。我在伪证责任下声明上述内容属实，并同意接受【适用法院/仲裁地】管辖。签名：*****；日期：*****。

在收到合规反通知后，平台通常会将该反通知副本发送给原投诉方，并告知对方平台可能在 **10 至 14 个工作日**（或平台规定的合理期限）后恢复该内容，除非原投诉方在该期限内向法院提起诉讼请求禁令（或其他法律救济）。具体恢复时点以平台审查与适用法律为准。

## 虚假或恶意声明的法律责任

提交虚假版权主张或恶意滥用下架通道可能会触发法律责任（例如在美国 DMCA 下，提交虚假声明者可能承担 §512(f) 的责任）。无论您身处何地，故意提交虚假声明并造成损害可能导致民事或刑事责任。平台保留对滥用投诉的应对权（包括拒绝受理、记录滥用并对发起人采取相应措施、以及在必要时保留追偿权）。

## 常见问答（FAQ）

* **Q：收到投诉后平台会马上永久删除吗？**
  A：平台在收到完整通知时会**先行移除或断开访问**，并在后续的人工/法律核验中决定是否永久删除或恢复（视反通知与法律程序而定）。

* **Q：我如何为我的作品提供证明？**
  A：建议上传权属证明（合同、授权邮件、购买发票、创作源文件、发行合同等），并在上传时在作品信息中填写权利人联系方式与授权声明。

* **Q：平台是否对版权争议承担责任？**
  A：平台会遵循现行法律与本政策及时处理投诉。平台并不在此承担超出法律规定的额外责任；若需进一步法律救济，请咨询律师或通过司法途径解决。
