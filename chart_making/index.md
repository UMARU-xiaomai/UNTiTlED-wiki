## 下载并运行制谱器以获取 UUID
您可以从[这里](/building.md)下载 未界音律UNTiTlED 制谱器。

将压缩文件解压至任意位置之后，双击 ChartEditorOfUNTiTlED.exe 运行制谱器。

在“设置”页面的“授权”栏目中，点击 “UUID” 项右侧的“拷贝”按钮来复制 UUID 。

## 上传您的谱面

当您完成谱面制作后，如果您想要将谱面上传到 UNTiTlED Community 并向所有 UNTiTlED 玩家分享您的谱面，请按照以下步骤：

### 一、确认 UNTiTlED Community 存在您的谱面使用的歌曲的记录

1. 输入歌曲的标题或者艺术家；
2. 单击“获取已记录的音乐”；
3. 在下面的视图中单击您的谱面对应使用的音乐；

::: warning
这将会**直接覆盖**您在“谱面信息”中输入的“标题”和“艺术家”信息，请确保选择的音乐信息与您在“谱面信息”中输入的完全一致。
:::

4. 检查“谱面信息”中“歌曲ID”一栏，确保其值不为0。

#### 如果找不到对应的音乐信息

请通过[上面的联系方式](#审核)联系我们添加。

### 二、确保谱面内容完整

一个完整的UBM谱面通常包含以下内容：

- 一个json谱面文件
- 一个mp3音频文件
- 一个jpg图片文件 （可选）

::: tip
jpg图片文件 作为封面，通常是可选的，但我们强烈推荐您添加。
:::

请确保在 制谱器 -> 设置 -> 设置路径 中选择了正确的文件。

### 三、上传谱面

单击“导出并上传UBM谱面文件”即可将导出的UBM文件上传到 UNTiTlED Community。

::: success
在下方您可以看到上传状态，success则表明上传成功。
:::

::: warning
为了避免恶意上传，谱面上传频率受到限制，每个拥有上传权限的谱师 仅能上传**1次/每小时**和**3次/每天**。若要通过游玩检查未完成的谱面，请参阅[手动导入UBM谱面文件](/building.md)。
:::

### 四、将您的谱面设为待审核（Pending）状态

上传成功后，您的谱面即刻开放下载，但由于状态在等待中（WIP），无法使用**排行榜**等功能（详细参阅“[社区谱面的状态](/building.md)”）。

请进入 UNTiTlED 游戏本体 -> Online Missions -> My Missions ，找到您想要进行审核的谱面，单击右侧的时钟图标，此时谱面进入 Pending 状态

### 五、等待

恭喜，您的谱面已成功进入待审核（Pending）阶段，请耐心等待，如果您的谱面拥有较高质量，便会变为已审核（Qualified），便可使用排行榜等功能
